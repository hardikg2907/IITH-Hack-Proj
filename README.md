#All-cam Livestreams

## Getting started
Clone this project on your local machine. Make sure you have node.js installed.
To run this project, follow the given steps in your terminal.

```bash
cd backend
npm i 
npm start
```
Then paste this url in the browser:
```
localhost:3000/login.html
```
